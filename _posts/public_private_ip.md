# 공인 IP vs 사설 IP

    공용이냐 개인이냐

- 공인 IP

  인터넷 사용자의 로컬 네트워크를 식별하기 위해 ISP (인터넷 서비스 공급자)가 제공하는 IP 주소

  전세계에서 유일한 IP 주소

  외부로 공개되어있기에 다른 PC 로부터 접근 가능

- 사설 IP

  로컬 네트워크에 할당된 IP 주소

  IPv4 주소 부족으로 인해 서브넷팅된 IP

  라우터에 의해 로컬 네트워크 상에 할당됨

  하나의 네트워크 안에서 유일

  외부에서 접근 불가능
