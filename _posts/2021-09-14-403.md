# IP

    네트워크 계층에서 사용하는 컴퓨터간 데이터 통신 프로토콜

인터넷 프로토콜(Internet Protocol) 의 줄임말

IP 주소를 가리키는 말로도 쓰임

OSI 7계층에서 네트워크 계층에서 사용하는 프로토콜

상위 계층에서 사용하는 TCP 와 엮여서 TCP-IP 로 표현하는 경우가 많음

호스트에서 호스트까지의 통신을 책임짐

IP 주소 는IP 통신에 필요한 고유 주소로 IPv4, IPv6 두가지 체계 존재
