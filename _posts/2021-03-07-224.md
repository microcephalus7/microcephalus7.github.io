# AWS CodeBuild

    컴파일부터 배포까지 하는 CI 도구

소스 코드를 컴파일하는 단계부터 테스트 실행 후 소프트웨어 패키지를 개발하여 배포하는 단계까지 마칠 수 이쓴ㄴ 완전관리형의 지속적 통합 서비스

자체 빌드 서버를 프로비저닝, 관리 및 확장할 필요가 없음

여러 빌드를 동시에 처리함

사전 패키징된 빌드 환경을 사용하면 신속하게 시작할 수 있음

자체 빌드 도구를 사용하는 사용자 지정 빌드 환경을 만들 수 있음
