# 쓰로틀링

    마지막 함수가 호출된 후 일정 시간이 지나기 전에 다시 호출되지 않도록 하는 것

`let timer; function () {if(!timer) {timer = setTimeout(function(){timer = null; console.log("타이머")})}}`

timer 가 설정되어 있다면 아무것도 하지 않음

timer 가 설동되어 있지 않으면 일정 시간 후에 스스로를 해제하며 특정 요청을 보냄
