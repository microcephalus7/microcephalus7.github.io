# Nginx 간단 정리

    동시접속 처리에 특화된 웹 서버

Apache 보다 단순

전달자 역할만 하기 떄문에 동시접속 처리에 특화

- 역할

  - 정적 파일을 처리하는 HTTP 서버로서의 역할

    HTML, CSS, JS, 이미지 등과 같은 정적 파일들을 웹 브라우저에 전송

  - 서버에 요청을 보내는 리버스 프록시로서의 역할

    가짜 서버를 두고 클라이언트의 요청을 서버에 배분하는 로드밸런서 역할을 함
