# 데이터의 이동과 변환을 자동화하는 웹 서비스

데이터 변환의 파라미터 정의 시 설정한 로직을 실행함

- 파이프라인 정의 파일 구문

  데이터 관리의 비즈니스 로직 지정

- 파이프라인 활성화

  Amazon EC2 인스턴스를 생성하여 작업을 예약하고 실행

  파이프라인 정의를 파이프라인에 업로드한 다음 파이프라인 활성화

  정의 편집의 효력은 파이프라인 대시 활성화 해야 함

- Task Runner

  작업에 대한 폴링을 수행한 다음 작업을 수행
