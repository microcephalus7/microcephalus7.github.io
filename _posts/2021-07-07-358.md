# Mongoose

    MongoDB 의 ODM 모듈

MongoDB 의 ODM node 모듈

- ODM (Object Document Mapping)

  객체(JS Object)와 문서(Document)를 1 대 1로 매칭

  문서를 DB 에서 조회 시 JS 객체로 바꿔주는 역할 함

- 기능

  Schema 부활, populate, 프로미스와 콜백 사용, 편리한 쿼리 빌더
