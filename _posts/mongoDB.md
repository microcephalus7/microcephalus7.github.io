# MongoDB

    문서 지향 저장소를 제공하는 NoSQL DB 시스템

문서 지향 저장소를 제공하는 NoSQL DB 시스템

- Document

  데이터의 명칭

  BSON 형태로 제공됨

- Collection

  Document 의 집합

  Schema 의 제약이 없음

- BSON

  Binary JSON 의 약자

  binary 로 인코딩된 JSON

  내장 문서와 배열 따위의 표현 가능

  복잡한 객체의 계층 관계를 하나의 레코드로 표현 가능

- 문서지향

  객체 지향 프로그래밍과 잘 맞음

  JSON 을 사용할 때 유용
