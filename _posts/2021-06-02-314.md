# 브라우저 이벤트

    무언가 일어났다는 신호

- 정의

    무언가 일어났다는 신호

- 왜 있음?

    UI 구성 (유저와 인터렉션 하기 위해)

- 어디서 일어남?

    DOM, 키보드, Document 등 

- 주체는?

    사용자, HTML, CSS, JS 등

- 언제 일어남?

    사용자의 마우스/키보드 조작, Document 의 문서 처리, JS 의 이벤트 발생 등

- 어떻게 일어나고 어떻게 처리함?

    1. 이벤트 캡쳐링

    2. 이벤트 핸들러 호출

    3. 이벤트 버블링
